# Timer 
### A rust CLI timer

Display a progress bar oin the terminal for the timer.
Availables units are : `h`, `m`, `s`. If empty, the default one will be seconds.

*Exemples :* 
```bash
$ timer <duration> <name?>
$ timer 10 ten-second-timer 
$ timer 10s ten-second-timer 
$ timer 7m pasta
$ timer 10h siesta
```
